# ESP32-Visualcar
DEMO中目前实现了利用手柄肩键实现对电机的无级调速，TFT实现了图片文字的显示

> **环境说明：**
>
> * 开发板：esp32dev
> * 开发环境：vscode+platformIO
>   
> **开发板说明：**
>
> * 电机驱动：TB6612
> * TFT驱动：ST7789_DRIVER
>   
---

> 使用到的开源项目和部分链接：
>
